# iOS Design Brief

## Intended Feel

- Single transcript-first screen
- Bottom-anchored composer
- Quiet chrome and generous spacing
- Warm neutral palette with restrained green accents
- Minimal motion, mostly for message arrival and scroll anchoring

## Allowed Primitives

- Rounded message surfaces
- Soft gradients and blurred ambient shapes
- System typography with a rounded treatment
- Inline error and retry affordances
- Empty-state suggestion prompts

## Do Not Copy

- No OpenAI marks, product names, or branding in the chrome
- No proprietary icons or exact app icon treatment
- No clone-level spacing, iconography, or visual details
- No settings-heavy UI that turns the app into something broader than chat

## Acceptance Check

If the screen reads as a calm, modern chat client without being mistaken for the real ChatGPT app, it is within bounds.
